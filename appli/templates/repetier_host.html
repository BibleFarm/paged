Impression 3D

Préparer l'impression 3D avec Repetier Host

Repetier Host est un slicer open source avec lequel on peut préparer l'impression 3D d'un objet.

Sélectionnez tout d'abord le profil de l'imprimante 3D que vous souhaitez utiliser en cliquant sur Réglages imprimante en haut à gauche. Pour importer un objet 3D, cliquez sur l'icône « + » entourée d'un cercle ou sur Charger en haut à gauche.

C'est également dans cet onglet que se font les réglages concernant la température de l'extrudeur ainsi que celle du lit. Dans Repetier Host, on ne définit pas le matériau qui va servir à l'impression, mais les temprétaures d'impression. Renseignez-vous donc sur les températures de l'extrudeur et du plateau propres à chaque matériau (en général indiquées sur la boîte du filament).

Dans l'onglet Placement d'objets du menu de gauche on trouve des outils permettant de redimensionner, de pivoter ou encore de prévisualiser le tranchage de l'objet.

Pour trancher l'objet, il faut se rendre dans l'onglet Trancheur. Vous pouvez sélectionner différents trancheurs dans le menu déroulant, mais Slic3r est un très bon trancheur et fonctionne très bien dans Repetier Host. Pour lancer le tranchage, cliquez sur le bouton Trancher avec Slic3r.

Une fois le tranchage fait on se retrouve dans l'onglet Pré visualiser impression, où on peut voir les couches de l'impression à venir. C'est ici qu'on peut enregistrer le fichier GCODE sur un périphérique.
