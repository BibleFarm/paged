Découpe laser

Exporter un fichier DXF depuis Freecad

Le format DXF est utilisé pour la découpe laser, il s'agit à la base d'un format propriétaire créé par Autodesk aujourd'hui tombé dans le domaine public. C'est un format de fichier 2D.

On peut dessiner une forme traçable par une découpeuse laser en créant un Sketch dans l'atelier Part design. On peut utiliser tous les outils de dessin disponibles ainsi que les contraintes si on veut pouvoir modifier le croquis paramétriquement.

Une fois l'esquisse terminée, cliquez sur Close. Pour exporter le fichier en DXF, sélectionnez l'esquisse dans l'arborescence du projet et cliquez sur Fichier > Exporter puis choisissez le format Autodesk DXF 2D.
