Arrondis et chamfreins dans Freecad

Les arrondis et les chamfreins permettent d'adoucir les angles d'objets modélisés en 3D. C'est particulièrement utile dans le cas d'objets à découper avec une CNC, afin de rendre l'objet fabriqué plus ergonomique.

Pour appliquer un arrondi sur une arête ou une face du solide, il faut la sélectionner en cliquant dessus, puis sélectionner l'outil Create fillet. Une boîte de dialogue s'ouvre dans le menu de gauche, vous y déterminez le rayon de l'arrondi.

Pour appliquer un chamfrein sur une autre arête, cliquez dessus puis sélectionnez l'outil Create chamfer. De la même façon que pour l'arrondi, la boîte de dialogue dans le menu vous permet de choisir la longueur du chamfrein.
